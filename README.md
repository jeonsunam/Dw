.red라는 클래스를 사용하면 바탕화면이 붉은색이 됩니다.

```html
<p class="red">배경이 붉은색이 됩니다.</p>
```

absolute position을 이용해서 중앙정렬하는 방법

```css
.center {
  position: absolute;
  top: 50%;
  left: 50%;
  transform: translate(-50%, -50%);
}
```
